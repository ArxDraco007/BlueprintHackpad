# ArxHackpad
ArxHackpad was made by Aryan India Gavaskar from Tamil Nadu, India. Fully modular with 4 Cherry MX style switch ports and 1 rotatory encoder. It uses the SEEED Studios XIAO RP2040 as the primary board.

This was my first time creating a good hardware project, which involved fusion and KiCad! I think it came out better than expected. It sucked up plenty of time, but it is worth the while.

BOM FILE - https://docs.google.com/spreadsheets/d/1qsAc0E0DF_lVhYs2crAtgfYjfSnlXe2ltoJdcgPbzX0/edit?usp=sharing

## BOM
1x PCB

1x Seeed XIAO RP2040

4x Cherry MX-style Switches

4x Cherry MX-style Keycaps (Blank)

1x EC11 Rotary Encoder

4x M3x16mm screws

4x M3x5mx4mm heatset inserts

1x Case to be entirely 3D-printed (self-provided); iterations may occur.
